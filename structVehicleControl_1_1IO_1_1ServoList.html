Detailed Description

The ServoList struct holds the names of all servos used in the application. This allows the array indexes to be accessed by name, not array position.

Note
In order to add servos to the app, add an enum here to describe the servo. Then, add the proper output construction in IOFactory.cpp's fillServoList method.
