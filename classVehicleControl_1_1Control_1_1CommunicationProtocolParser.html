Detailed Description

The Control::CommunicationProtocolParser class handles parcing and packing communication packets. This class is the only one that "knows" how the protocol is structured and packed. Therefore, this class, for instance, sets each servo to its desired value. Additionally, this class packs up all the info that is sent to the tablet.
