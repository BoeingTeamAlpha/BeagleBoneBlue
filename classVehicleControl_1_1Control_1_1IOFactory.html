Detailed Description

The Control::IOFactory class creates all of the app's IO. Therefore, EVERY IO pin's creation must be defined in its respective method. This class just acts as a "factory" for the Control class to create the IO upon initialization.

Note
This class is the only object that "knows" how to create the app's IO, so every IO point must be defined here.
